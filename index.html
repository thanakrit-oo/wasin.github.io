ระบบยืนยันตัวตน Discord

กรอกข้อมูลเพื่อสร้างรหัสยืนยันอย่างปลอดภัย แล้วใช้กับบอทของเรา

ขั้นตอนการยืนยัน

  1. กรอกชื่อ Discord ของคุณในช่องด้านล่าง
  2. เลือกช่องทางที่รู้จักเซิร์ฟเวอร์ของเรา
  3. กดปุ่ม "สร้างรหัสยืนยัน"
  4. คัดลอกรหัสที่ปรากฏ และนำไปใช้กับคำสั่ง /verify key:รหัส ใน Discord
  5. คุณจะได้รับยศทันที!

หมายเหตุ: ระบบบอทยอมรับได้ทั้ง Username, Display name และ Global name
